## 一、Ubuntu / Debian 系工具

| 工具或命令                     | 说明                  |
|---------------------------|---------------------|
| `apt`, `apt-get`          | Debian 包管理器（替代 yum） |
| `/etc/network/interfaces` | 网络配置文件（传统方式）        |
| `ufw`                     | Ubuntu 自带的简化防火墙     |
| `snap`                    | 新的通用软件包格式           |
| `add-apt-repository`      | 添加软件源               |
| `update-alternatives`     | 多版本管理，如 Java        |
