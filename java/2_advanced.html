2、流式API（Stream API）与集合框架的结合

Java 8引入的Stream API提供了一种以声明式方式处理集合的能力。Stream API提供了丰富的操作链式调用，如map、filter、reduce 等，能够使得代码更加简洁和具备函数式编程风格。

  • 流的种类：详情见：Java总结-Java：说说Java的stream

  • 集合框架与流结合：

    • Collection.stream()：从集合（如List、Set）中创建流。
    • Stream.collect()：用于将流的结果收集到集合中，支持自定义收集器。
    • Stream.forEach()：对流中的每个元素执行操作。
